## 📝 Example

### Question

What is the difference between rasterization and ray tracing?

### Answer

* **Rasterization** converts geometric primitives into pixels by projecting them onto the screen. It is fast and widely used in real-time rendering.
* **Ray tracing** simulates the physical behavior of light by tracing rays, producing more realistic images but at a higher computational cost.
